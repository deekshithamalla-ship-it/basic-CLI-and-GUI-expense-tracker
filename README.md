# basic-CLI-and-GUI-expense-tracker

This project shows my learning journey from basic Python to GUI development.

##  Version 1: Basic (CLI)
- Used lists and dictionaries
- Input/output using terminal
- Features:
  - Add expense
  - View expenses
  - Total expenses
  - Category-wise expenses

##  Version 2: GUI (Tkinter)
- Built a graphical interface using Tkinter
- Improved user experience
- Features:
  - Add expense using form
  - View in text area
  - Total calculation
  - Category-wise summary

##  Future Improvements
- Save data to CSV file
- Add delete/edit option
- Better UI design
